# Partner Logos

Place all marketplace/partner logos for the homepage marquee in this folder.

## Required filenames

Use these exact filenames so the site picks them up automatically:

- `smashwords.png`
- `barnes-and-noble.png`
- `google-books.png`
- `draft2digital.png`
- `amazon.png`
- `kindle.png`

## Suggested logo format

- Transparent PNG or SVG converted to PNG
- Minimum width: 280px
- Recommended canvas ratio: around `3:1` to `4:1`
- Keep equal top/bottom spacing so logos appear visually aligned in the marquee

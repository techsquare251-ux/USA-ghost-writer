# Social Media Logos

Place all social media brand logos used by the footer in this folder.

## Required filenames

Use these exact filenames so the footer picks them up automatically:

- `facebook.png`
- `linkedin.png`
- `x.png`
- `instagram.png`
- `youtube.png`

## Suggested logo format

- Preferred: transparent PNG
- Recommended size: at least 96x96
- Keep logos centered with consistent whitespace
- Use monochrome or clean official marks for a professional look
